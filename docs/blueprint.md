# **App Name**: ServicePulse Dashboard

## Core Features:

- Revenue Statistics Cards: Display key revenue metrics such as total revenue, today's revenue, total business, and other relevant stats using visually appealing cards.
- Revenue Chart: Visualize revenue trends over time using an interactive chart.
- Low Stock Items Table: Show a table listing items that are low in stock, allowing for quick reordering.
- Recent Invoices Table: Display a table of recent invoices with key details like invoice number, customer, date, and amount.

## Style Guidelines:

- Primary color: Dark blue (#1A237E) to convey professionalism and trust.
- Background color: Light gray (#F5F5F5), creating a clean and modern backdrop.
- Accent color: Teal (#008080) for highlighting important elements and CTAs.
- Body and headline font: 'Inter' sans-serif font for a modern and readable experience.
- Responsive layout adapting to different screen sizes, ensuring optimal viewing on mobile and desktop devices.
- Use a set of clear, professional icons to represent different data points and actions.
- Subtle transitions and animations to improve user engagement without being distracting.
